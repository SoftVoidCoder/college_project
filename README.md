# Точка

Минималистичное локальное приложение для заметок с тегами, поиском и фильтрацией. Данные остаются в браузере и не отправляются на сервер.

## Запуск

```bash
python3 -m http.server 8090 --bind 127.0.0.1
```

Откройте [http://127.0.0.1:8090](http://127.0.0.1:8090).

## Структура

```text
index.html                  точка входа
src/scripts/app.js          управление интерфейсом
src/scripts/domain/         правила работы с заметками
src/scripts/services/       локальное хранилище
src/styles/                 стили по зонам ответственности
docs/                       документация и материалы
```

При установленном Node.js доступны короткие команды `npm start` и `npm run check`.
